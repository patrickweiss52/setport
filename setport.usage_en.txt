
Usage: {OPTION}... {PORT}...
Listen for the specified port; must be contained in the range 0 < x < 65536

Mandatory arguments to long options are mandatory for short options too.
-p, --port opens the stream to read in desired port
-?,-h,--help displays the usage file for setport
-!, --about displays the about information
-v, --version displays the current version of setport
